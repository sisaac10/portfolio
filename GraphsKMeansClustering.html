Matplotlib Graphs and K-Means Clustering Algoritm¶

Implement k- means algorithm for Wisconsin Breast Cancer Dataset:

https://archive.ics.uci.edu/ml/machine-learning-databases/breast-cancer-wisconsin/breast-cancerwisconsin.data

  1. Plot nine histograms for attributes A2 to A10
  2. Find the mean, median, standard deviation, and variance of each of the attributes A2 to A10
  3. Save plots to pdf file
  4. Out of input 699 datapoints, select any 2 random datapoints as initial mean resulting into 2 nine dimensional vectors (clusters)
  5. Compute each of the euclidian distance between each of the 699 datapoints and the two initial means
  6. Assign datapoint to the cluster that is the shortest distance
  7. Iterate until convergence (all datapoints are assigned to one of two clusters)
  8. Calculate error rate for each cluster (ratio of correctly predicted classes)

import matplotlib.pyplot as plt

def histo(seri, titl, clr):                        #generates plots with mean, median, standard dev, and variance 
    
    mn = round(seri.mean(),2)
    md = round(seri.median(),2)
    sd = round(seri.std(),2)
    v = round(seri.var(),2)
    mn = str(mn)
    md = str(md)
    sd = str(sd)
    v = str(v)
    tx = titl + " -  Mean: " + mn + ", Median: " + md + "\n Standard Deviation: " + sd + ", Variance: " + v
    fig = plt.figure(figsize=(7,4))
    hw1 = fig.add_subplot(1,1,1)
    hw1.hist(seri, bins = 18, color = clr) 
    hw1.set_title(tx)
    hw1.set_ylabel('Number of Instances')
    hw1.set_xlabel('Rating 1-10')
    plt.setp(hw1.get_yticklines(), visible = False)
    plt.setp(hw1.get_xticklines(), visible = False)

    mu2[6] = blandchrom[u2]

    mu4[1] = unicsize[u4]
    mu4[2] = unicshape[u4]
    mu4[3] = margadh[u4]
    mu4[4] = singepicsize[u4]
    mu4[5] = barenuc[u4]
    mu4[6] = blandchrom[u4]
