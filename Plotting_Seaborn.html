Plotting Data with Seaborn¶

Seaborn is a Python visualization library based on matplotlib. It provides an interface for drawing statistical graphics and it includes support for numpy and pandas.

You can use the PairGrid() function to create a grid of subplots to plot relations between pairs of variables. On the diagonal of the grid, you can plot the KDE of each variable using the map_diag() method. On the off-diagonal subplots, you can plot the 2-D KDE of pairs of variables using the map_offdiag() method.

In [16]:
g = sns.PairGrid(iris)
g.map_diag(sns.kdeplot)
g.map_offdiag(sns.kdeplot, n_levels=5) # set the number of contour levels to 5

You can vary the number of plots and their location. For example, you can plot the KDE on the diagonal; plot a scatter plot between two variables on the lower diagonal subplots; and on the upper diagonal subplots you can plot a 2-D KDE of two variables.

In [17]:
g = sns.PairGrid(iris,vars=["sepal_length", "sepal_width"])
g.map_diag(sns.kdeplot)
g.map_upper(sns.kdeplot, n_levels=5)
