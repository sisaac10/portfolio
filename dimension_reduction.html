Dimension reduction with t-SNE¶

Visualizing high-dimensional data is an important problem. Some data sets can contain thousands of high-dimensional datapoints. Visualizing a data set with this many dimensions can result in something that is very confusing for your intended audience. In order to visualize this information you have to convert a high-dimensional data set into a 2 or 3 dimensional data set. We will look at t-SNE, a dimensionality reduction method that can help us achieve this goal. Dimensionality reduction is used to preserve as much of the meaningful structure of the high-dimensional data as you convert it to a low-dimensional map.

t-SNE is a method for exploring high-dimensional data."The goal is to take a set of points in a high-dimensional space and find a faithful representation of those points in a lower-dimensional space, typically the 2D plane. The algorithm is non-linear and adapts to the underlying data, performing different transformations on different regions." - How to Use t-SNE Effectively

To run t-SNE, you should first reorder the data points according to the handwritten numbers. You can use np.vstack and np.hstack.

In [22]:
X = np.vstack([digits.data[digits.target==i]
               for i in range(10)])
y = np.hstack([digits.target[digits.target==i]
               for i in range(10)])

Then initialize a t-sne model. For the meaning of the parameters see here.

In [23]:
tsne = manifold.TSNE(n_components=2, init='pca', random_state=0)

Fit the model on the data.

In [24]:
digits_proj = tsne.fit_transform(X)

t-SNE clearly seperates the digits into clusters. You can add some text to each cluster, with the placement of the text being at the center of the cluster. You can use np.median to find the centers. To make it more convenient, you can make the code into a function.

In [27]:
def plot_scatter(projection):
    plt.figure(figsize = (6,6))
    plt.scatter(projection[:,0], projection[:,1],c=palette[y])
    for i in range(10):
        # Position of each label.
        xtext, ytext = np.median(projection[y == i, :], axis=0)
        txt = plt.text(xtext, ytext, str(i), fontsize=24)
